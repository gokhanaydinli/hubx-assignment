# HubX Frontend Assignment

A responsive, mobile-first slider component showcasing document scanning features.

## Tech Stack

- React 18.2.0
- TypeScript 5.1.6
- Framer Motion 12.23.26
- Swiper 11.1.15
- Vite 6.0.11

## Getting Started

Install dependencies:

```sh
npm install
npm install react react-dom
npm install swiper
npm install framer-motion
npm install -D typescript vite
```

Start development server:

```sh
npm run dev
```

## Code Quality

Lint the code:

```sh
npm run lint
```

Format the code:

```sh
npm run format
```

Build for production:

```sh
npm run build
```
